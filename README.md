# Alpha Landing Page

This folder is a static GitHub Pages version of the landing page.

## Files

- `index.html`
- `styles.css`

## Publish with GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Set the source to the branch you want to publish from.
4. If this repo contains only the landing page, publish from `/ (root)`.
5. If this repo is a larger repo, point GitHub Pages to this folder or move these files into the publish root.

## Before going live

- Replace the placeholder form with Tally, Formspree, Typeform, or your own backend.
- Update copy in `index.html` as needed.
